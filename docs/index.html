Three commands, one border crossing

How it clears customs

01

Probe

Inspect a local file or URL — duration, resolution, captions, any sidecar transcript. No cost yet.

video.py probe clip.mp4
02

Estimate

Price the whole job up front — tokens, dollars, whether a model needs downloading, whether any backend in the chain would leave the machine.

video.py estimate clip.mp4 --human
03

Run

Only after approval. Cloud backends and one-time model downloads each need their own explicit flag — an API key sitting in the environment is never treated as consent.

video.py run clip.mp4 --allow-cloud

The point of the whole tool

What's actually gated

$

Priced before it runs

estimate computes the full token + dollar cost — frames, transcript, output — before run touches anything.

⌂

Local by default

captions/sidecar/faster-whisper/trx never leave the machine. Cloud is opt-in, never assumed.

✓

Consent, not a key check

--allow-cloud and --allow-model-download are required per invocation. A key in the environment isn't approval.

↺

Rejected before conversion

A disallowed cloud chain is refused before the file is even downloaded or converted to audio.

Zero keys, zero cost, one clone
